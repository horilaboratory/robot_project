# 0. Ubuntu 22.04 を起動する方法、シャットダウンする方法
　WSL2 内にインストールした Ubuntu 22.04 を起動する方法を解説します。

> [!CAUTION]
> ## チュートリアルを進める前に ...
> ドキュメント [Windows でロボット開発をはじめる](/windows/hostsetup.md) を読んでコンピューター上に必要な環境を構築しましたか？

## Ubuntu 22.04 の起動方法
　スタートメニューを開き、検索バーに **「terminal」** または **「ターミナル」** と入力して、表示された「ターミナル」アプリを起動します。

<img src="/imgs/open_terminal.png" width=40% /><img src="/imgs/opened_terminal.png" width=60% />
 
　ターミナル上部タブにある「+」ボタン横にある下向き矢印アイコンをクリックして、**「Ubuntu 22.04」** をクリックしてください。
すると紫色のターミナルが表示されます。紫色のターミナルが **Ubuntu 22.04** です。

<p align="center">
<img src="/imgs/terminal1.gif" width=80% />
</p>

> [!TIP]
> ターミナル起動時、デフォルトで Ubuntu 22.04 が起動するように設定したい場合
> [**デフォルトを Ubuntu にする方法**](/windows/terminal.md/#editdefault)
> を参照してください。

## Ubuntu 22.04、WSL2 のシャットダウン方法
　Ubuntu 22.04 は WSL2 環境下で仮想環境として動作しています。Ubuntu 22.04 をシャットダウンするには **WSL2 本体をシャットダウンさせる** 必要があります。

> [!TIP]
> ### 何のために WSL2 をシャットダウンさせるか
> 　以下の作業やトラブルに見舞われた場合 WSL2 をシャットダウンすべきです。
> - WSL2 設定ファイル `.wslconfig` を編集した場合
> - Ubuntu 22.04 がフリーズした場合
> - Ubuntu 22.04 が多くのメモリ、CPU を消費しホスト（Windows側）が大きな負荷を負っている場合
> - ホストで動画編集やゲームなど重たい作業をしたい場合
> - バッテリーの消費を節約したい場合
>
> WSL2 が起動すると、バックグラウンド上で Ubuntu 22.04 が動作し続けます。つまり **2つの OS が動いている** 状態が続くのです。バッテリーを節約したい場合は WSL2 をシャットダウンさせておくと良いでしょう。WSL2 は、一度シャットダウンされると PowerShell 上で起動コマンドを打つ。または直接 Ubuntu 22.04 をターミナルから立ち上げない限り起動しません。

　ターミナルを開き、**PowerShell** を起動してください。もしターミナルのデフォルトインターフェースを Ubuntu 22.04 にしている場合はターミナル上部タブにある「+」ボタン横にある下向き矢印アイコンをクリックして、**「Windows PowerShell」** をクリックしてください。
<br>
　PowerShell 上で以下のコマンドを実行して WSL2 をシャットダウンさせます。
```powershell
wsl2 --shutdown
```
WSL2 がシャットダウンすると使用していた Ubuntu 22.04 は停止し、Ubuntu 22.04 ターミナルで以下のメッセージが表示されます。Ubuntu 22.04 のタブを閉じるかキーボードショートカット「Control + D」を実行して終了することができます。

> [!NOTE]
> この状態でエンターキーを押すと Ubuntu が再起動します。
